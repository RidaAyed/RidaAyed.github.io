GPG export your ssh authentication keys

8 September, 2021

First get our authentication [A] subkey’s fingerprint:

gpg2 --list-secret-keys --keyid-format LONG | grep '\[A\]'

ssb   rsa4096/45EDA5801C0D18FF 2019-04-05 [A]

Now we use this subkey’s fingerprint to generate both our private and public ssh keys:

temp=/tmp/id_rsa #Temp file
rm -f $temp #Delete previous temp file
public=~/.ssh/id_rsa.pub
private=~/.ssh/id_rsa
gpg2 --export-ssh-key 45EDA5801C0D18FF > $public
gpg2 --armor --export-secret-subkeys 45EDA5801C0D18FF > $temp
chmod 400 $temp
ssh-keygen -y -e -f $temp > $private

tail $private -n 3 #Show us some lines after finishing
